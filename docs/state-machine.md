# VoiceInput 状态机设计

v1


| 状态              | 面板提示栏                            | 触发键                   | 回车 (⏎)                | ESC                   | F2                  | 点击面板                |
| --------------- | -------------------------------- | --------------------- | --------------------- | --------------------- | ------------------- | ------------------- |
| **状态一：空闲**      | 无面板                              | 开始录音，显示面板 → **状态二**   | —                     | —                     | —                   | —                   |
| **状态二：录音中**     | `ESC : 退出 ⏎/快捷键 : 插入 点击/F2 : 编辑` | 停止录音，插入文字 → **状态一**   | 停止录音，插入文字 → **状态一**   | 取消，关闭面板，不插入 → **状态一** | 停止录音，进入编辑 → **状态三** | 停止录音，进入编辑 → **状态三** |
| **状态三：编辑模式**    | `ESC : 取消 快捷键 : 追加识别 ⏎ : 确认插入`   | 开始追加识别 → **状态四**      | 确认插入到目标应用 → **状态一**   | 取消，关闭面板，不插入 → **状态一** | —                   | —                   |
| **状态四：编辑+追加识别** | `快捷键 : 停止识别 ESC : 取消识别 ⏎ : 确认插入` | 停止追加识别，保持编辑 → **状态三** | 停止追加识别，确认插入 → **状态一** | 停止追加识别，保持编辑 → **状态三** | —                   | —                   |


v2


|         |                                 |                       |                 |                 |                   |                  |
| ------- | ------------------------------- | --------------------- | --------------- | --------------- | ----------------- | ---------------- |
| 状态      | 面板状态                            | 面板提示栏                 | 触发键             | 回车 (⏎)          | ESC               | 点击面板             |
| 状态一：空闲  | 无面板                             | 无面板                   | 开始录音，显示面板 → 状态二 | —               | —                 | —                |
| 状态二：录音中 | 编辑状态。在光标处可以输入文字。新的识别结果也直接显示在光标处 | ESC : 退出 ⏎/快捷键 : 结束输入 | 停止录音，插入文字 → 状态一 | 停止录音，插入文字 → 状态一 | 取消，关闭面板，不插入 → 状态一 | 保持状态二。允许点击改变光标位置 |


v3

| 状态 | 面板状态 | 面板提示栏 | 触发键 | 回车 (⏎) | ESC | 点击面板文字区 | 点击"重新识别"按钮 |
|------|---------|-----------|--------|-----------|-----|-------------|-----------------|
| **状态一：空闲** | 无面板 | — | 开始录音 → **状态二** | — | — | — | — |
| **状态二：录音中** | 可编辑，识别结果插入光标处 | `ESC : 取消  ⏎/快捷键 : 结束输入` | 停止录音，插入文字 → **状态一** | 停止录音，插入文字 → **状态一** | 取消，关闭面板，不插入 → **状态一** | 停止录音，光标定位到点击处 → **状态三** | — |
| **状态三：编辑中** | 可编辑，显示"重新识别"按钮 | `ESC : 取消  ⏎ : 确认插入` | — (不响应) | 确认插入到目标应用 → **状态一** | 取消，关闭面板，不插入 → **状态一** | 改变光标位置，保持 **状态三** | 从光标处开始追加识别 → **状态二** |


v4

| 状态 | 面板状态 | 面板提示栏 | 触发键 | 回车 (⏎) | ESC | 点击面板文字区 | 点击"继续识别"按钮 |
|------|---------|-----------|--------|-----------|-----|-------------|-----------------|
| **状态一：空闲** | 无面板 | — | 开始录音 → **状态二** | — | — | — | — |
| **状态二：录音中** | 可编辑，识别结果插入光标处 | `ESC : 取消  ⏎/快捷键 : 结束识别` `点击文字进行编辑` | 停止录音，插入文字到目标应用 → **状态一** | 停止录音，插入文字到目标应用 → **状态一** | 取消，关闭面板，不插入 → **状态一** | 停止录音，光标定位到点击处 → **状态三** | — |
| **状态三：编辑中** | 可编辑，显示"继续识别"按钮 | `ESC : 取消  ⏎ : 确认插入` tip：ESC取消后可以到识别历史中查看、复制识别结果 | 确认插入到目标应用 → **状态一** | 确认插入到目标应用 → **状态一** | 取消，关闭面板，不插入 → **状态一** **但是识别结果要记录到历史中** | 改变光标位置，保持 **状态三** | 从光标处开始追加识别 → **状态二** |
